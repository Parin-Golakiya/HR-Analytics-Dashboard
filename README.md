# HR Analytics Dashboard
A comprehensive Tableau analytics solution designed to empower HR managers with actionable insights into workforce composition, demographics, compensation trends, and employee records. This dashboard transforms complex HR data into clear, actionable visualizations for strategic human resource management.

---

## üìä Project Overview
This HR Analytics Dashboard provides a complete view of organizational human capital through interactive visualizations and detailed employee records. The dashboard is structured into two main views:

Summary View: High-level insights across Overview, Demographics, and Income Analysis
Employee Records View: Detailed, filterable employee database for granular analysis

Built with HR managers in mind, this tool enables data-driven decisions on hiring, retention, compensation equity, and workforce planning.

- Salary comparison across education levels for both genders.
